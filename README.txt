Proyecto Validación de Registro

En este proyecto haremos una validación de regstro de usuario. Donde tendrá que llenar diferentes campos como el nombre, apellidos, correo, número de teléfono y password. 

Y en cada campo estaré utilizando las expresiones regulares para validar los diferentes campos.

Estarémos utilizando HTML para el maqueteado del registro, CSS para el estilo del registro y JavaScript lo usuaré para mandejar los eventos y guradar los datos del usuario en un objeto para poder manipularlo.
